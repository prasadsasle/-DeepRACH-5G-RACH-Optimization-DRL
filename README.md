# DeepRACH: 5G RACH Optimization using DRL 🚀

## 📌 Overview

DeepRACH is a Deep Reinforcement Learning-based framework designed to optimize Random Access Channel (RACH) performance in 5G networks under mixed traffic conditions.

It dynamically manages:

* Downlink bandwidth allocation
* Uplink preamble allocation

for efficient handling of:

* H2H (Human-to-Human)
* M2M (IoT devices)

---

## 🧠 Algorithms Implemented

* DQN (Deep Q Network)
* LSTM (Sequential learning for temporal traffic)
* A2C (Advantage Actor-Critic)
* PPO (Proximal Policy Optimization)

---

## ⚙️ Key Features

* Multi-agent DRL framework
* GAN-based data augmentation
* Lyrebird Optimization (LBO) for hyperparameter tuning
* 3GPP-compliant 5G simulation
* Interactive dashboard (Gradio)

---

## 📊 Results

* High access success rate (~0.88)
* Near-zero collision rate (with GAN)
* Fair resource allocation (Jain Index ≈ 1.0)
* Better performance than baseline RACH methods
